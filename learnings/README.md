# Learnings

This folder captures synthesized learning, not raw copied course notes.

Learning is secondary to the starter kit. A note belongs here only if it helps
improve how future projects are started, built, reviewed, deployed, or
evaluated.

Each note should answer:

- What did I learn?
- Why does it matter for real AI engineering?
- How will it change my workflow?
- What should I build or test next?
- What template, checklist, or project default should change?
